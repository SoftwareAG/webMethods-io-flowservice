Controls in debug toolbar

We have a toolbar in Debug Panel. It has different debug options we can use while debugging a FlowService.

Debug toolbar controls

search in the inputs

Disable/Enable breakpoints

Resume to next breakpoint

Stop the debug execution

Restart the execution

Stepover into the next flow

Clear all breakpoints

Close the debug panel

The above steps shows the different debug options available on the debug toolbar

Key shortcuts

When you hover on every control, the shortcut key for invoking the respective control is shown in tooltip.

ActionKey shortcut
Disable/Enable breakpointsCTRL + F8
ResumeF8
StopSHIFT + F5
RestartF5
Step overF10
Clear all breakpointsALT + F8
Close debug panelCTRL + Q

Note : If we invoke Clear all breakpoints action, we will not be able retrieve breakpoints back using undo action.
