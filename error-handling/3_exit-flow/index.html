How to exit FlowService on success or failure ?

Edit on GitHub

The Exit step exits the entire FlowService and signals success or failure as part of the exit.

FTP CSV to Salesforce Account

Get file from FTP

Convert output stream from FTP to document

Create Salesforce account from that document

If Create Account gives result as null

Exit the FlowService with failure message

If the result is not null

Log the message

Exit the FlowService with success
